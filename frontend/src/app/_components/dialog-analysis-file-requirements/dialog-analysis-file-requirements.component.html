Locality School School History file requirements

  • The supported CSV file type is comma separated values.
  • The CSV file should be in UTF-8 format.
  • Make sure to include a header with the column names in the first line of the file.
  • For Yes/No fields you can use this binary value: True/False.
  • For string columns, the value must be enclosed in double quotes.
  • For array[] columns, the value must be enclosed in double quotes and square brackets.
